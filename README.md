# StormDash

A unique dashboard to show simple alerts

## Getting Started

### Clone this repo

```bash
> git clone https://github.com/globocom/stormdash.git
> cd stormdash
```

### Install Dependencies

```bash
> npm install
```

### Build

```bash
> npm run build
```

### Start the Server

```bash
> npm run start:server
```
